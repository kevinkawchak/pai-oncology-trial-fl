v0.5.0 - Domain Examples and Engineering Demonstrations

## Summary

Adds 17 production-quality example scripts across three domains: 5 core federated learning examples, 6 physical AI robotics examples, and 6 agentic AI examples. Each script is 800-1300+ lines of self-contained, executable Python demonstrating end-to-end workflows with realistic clinical parameters, structured logging, and RESEARCH USE ONLY disclaimers throughout.

## Features

- **Federated Training Workflow** (`examples/01_federated_training_workflow.py`) — End-to-end federated learning pipeline demonstrating multi-site coordinator setup, FedAvg/FedProx aggregation, differential privacy integration, secure aggregation, and convergence monitoring across simulated hospital nodes
- **Digital Twin Planning** (`examples/02_digital_twin_planning.py`) — Patient-specific digital twin construction from synthetic imaging data with tumor growth modeling (exponential, logistic, Gompertz), treatment response simulation, and multi-organ physiology tracking
- **Cross-Framework Validation** (`examples/03_cross_framework_validation.py`) — Policy validation across Isaac Sim, MuJoCo, PyBullet, and Gazebo with kinematic drift analysis, force comparison, and cross-engine agreement metrics
- **Agentic Clinical Workflow** (`examples/04_agentic_clinical_workflow.py`) — Multi-agent clinical decision support with oncologist, radiologist, safety monitor, and data manager agents coordinating via unified interface with human-in-the-loop checkpoints
- **Outcome Prediction Pipeline** (`examples/05_outcome_prediction_pipeline.py`) — ML pipeline for treatment outcome prediction using federated features with survival analysis, biomarker integration, and model fairness auditing across demographic subgroups
- **Real-Time Safety Monitoring** (`examples-physical-ai/01_realtime_safety_monitoring.py`) — Real-time force/torque monitoring for surgical robotics with configurable safety thresholds, emergency stop logic, workspace boundary enforcement, and ISO 13482 compliance checking
- **Multi-Sensor Fusion** (`examples-physical-ai/02_multi_sensor_fusion.py`) — Kalman filter and complementary filter sensor fusion for surgical navigation combining optical tracking, IMU, force/torque, and stereo vision with latency compensation
- **ROS2 Surgical Deployment** (`examples-physical-ai/03_ros2_surgical_deployment.py`) — ROS2-based surgical robot deployment pipeline with lifecycle node management, QoS configuration, URDF loading, trajectory planning, and real-time control loop integration
- **Hand-Eye Calibration** (`examples-physical-ai/04_hand_eye_calibration.py`) — Automated hand-eye calibration for surgical robotic systems using Tsai-Lenz method with fiducial detection, reprojection error analysis, and calibration validation
- **Shared Autonomy Teleoperation** (`examples-physical-ai/05_shared_autonomy_teleoperation.py`) — Shared autonomy framework for surgical teleoperation with variable autonomy blending, operator intent prediction, haptic feedback generation, and safety-constrained execution
- **Robotic Sample Handling** (`examples-physical-ai/06_robotic_sample_handling.py`) — Automated tissue sample handling for biopsy workflows with grasp planning, force-controlled manipulation, contamination avoidance, and chain-of-custody tracking
- **MCP Oncology Server** (`agentic-ai/examples-agentic-ai/01_mcp_oncology_server.py`) — Model Context Protocol server implementation for oncology with tool registration (patient lookup, drug interaction checking, literature search), resource management, and PHI-safe prompt handling
- **ReAct Treatment Planner** (`agentic-ai/examples-agentic-ai/02_react_treatment_planner.py`) — ReAct (Reasoning + Acting) agent for treatment planning with thought-action-observation loops, NCCN guideline retrieval, drug interaction checking, and structured treatment plan generation
- **Real-Time Adaptive Monitoring Agent** (`agentic-ai/examples-agentic-ai/03_realtime_adaptive_monitoring_agent.py`) — Streaming monitoring agent that adapts alert thresholds based on patient acuity, tracks adverse events, and generates real-time clinical dashboards with trend analysis
- **Autonomous Simulation Orchestrator** (`agentic-ai/examples-agentic-ai/04_autonomous_simulation_orchestrator.py`) — Agent that autonomously manages digital twin simulation campaigns: parameter sweep design, job scheduling across simulation engines, result aggregation, and convergence detection
- **Safety-Constrained Agent Executor** (`agentic-ai/examples-agentic-ai/05_safety_constrained_agent_executor.py`) — Agent execution framework with multi-layer safety constraints: PHI detection on all I/O, action allowlisting, human-in-the-loop gates for high-risk decisions, and complete audit trail
- **Oncology RAG Compliance Agent** (`agentic-ai/examples-agentic-ai/06_oncology_rag_compliance_agent.py`) — Retrieval-augmented generation agent for regulatory compliance with FDA guidance retrieval, protocol deviation detection, IRB submission assistance, and citation-backed responses

## Notes
