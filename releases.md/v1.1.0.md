v1.1.0 - Federated Learning Physical AI Oncology Trials Unification Paper

## Summary

Release v1.1.0 introduces the first academic paper for the PAI Oncology Trial FL platform: "Federated Learning Physical AI Oncology Trials Unification." The 18-page paper, generated primarily by Claude Code Opus 4.6 from a hand-written prompt by Kevin Kawchak, describes the full architecture and end-to-end workflow demonstrations across the platform's five infrastructure pillars: Privacy Infrastructure, Regulatory Infrastructure, Cross-Framework Unification, Standards & Benchmarking, and Multi-Organization Cooperation. The paper includes quantitative codebase data, code snippets, architecture diagrams, and comprehensive analysis of the triple AI peer review process (v0.9.4–v0.9.9). DOI: 10.5281/zenodo.18795507.
